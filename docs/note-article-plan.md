### 4. MOC自動生成 — ノート整理の「本丸」
**文字数**: 800〜1,000字

#### 4-1. 3つのMOC作成モード

| モード | 特徴 | おすすめユーザー |
|--------|------|------------------|
| テンプレートモード | 5種類のテンプレートから選択 | 素早くMOCを作りたい人 |
| ウィザードモード | ステップバイステップで案内 | MOC初心者 |
| スマート提案モード | AI がキーワード/タグから関連ノートを自動提案 | 大量ノート持ち |

**解説ポイント**:
- 標準MOC / フォルダMOC / Zettelkasten MOC / タグベースMOC / カスタム
- キーワードベースの自動分類ルール（AI・LLM、マーケティング、開発・技術など）
- 既存MOCの更新機能：新しいノートを追加したら、MOCも自動更新
- 一括MOC生成：全フォルダに対してMOCをまとめて作成

**🖼️ 画像⑧: MOC作成ウィザード画面**
- ステップ形式のUIが表示されている状態
- テンプレート選択 → 基本情報 → プレビュー の流れが分かる

**🖼️ 画像⑨: 生成されたMOCのプレビュー**
- Obsidian上で実際に開いたMOCノートのスクリーンショット
- Dataviewクエリ付きの洗練されたMOC

#### 4-2. スマート提案の仕組み

- タグ入力 → 該当ノートを自動抽出
- フォルダ指定 → フォルダ内ノートを自動抽出
- MOC名のキーワード → ファジーマッチングで関連ノートを検索
- 3つの結果を統合してスコアリング → 最適な関連ノートをTOP N提案
